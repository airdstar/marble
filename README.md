# Description
Inspired by the Wii Fit marble game and Wariowarelike games
# Controls
## Keyboard/Mouse
Tilt: Mouse Movement<br/>
Camera Rotation: A/D<br/>
Close Game: Esc<br/>
## Controller Controls
Tilt: Left Stick<br/>
Camera Rotation: Right Stick<br/>
Close Game: Start/Pause<br/>
## General Controls
Open Settings: Tab
